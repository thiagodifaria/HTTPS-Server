🔧 Sobre o HTTPS Server

Detalhes técnicos e arquitetura do servidor de alta performance

⚙️

Stack Tecnológico

C++17 moderno com recursos avançados
OpenSSL 3.0 para criptografia e TLS
CMake sistema de build cross-platform
NASM assembler x86-64

🔐

Assembly Otimizado

AES-NI implementação hand-optimized
SHA-256 AVX hash com registradores AVX
Base64 SIMD encoding/decoding vetorizado
UUID v4 geração com hardware RNG

🚀

Performance

3.51 GB/s throughput AES-NI
HTTP Parsing acelerado com AVX2
Hex Encoding vetorizado
Validação SIMD de entrada JSON

📊 Benchmarks

AES-NI Assembly: 3.51 GB/s (20M blocos, 305 MB em 0.08s) SHA-256 Assembly: Alta performance com registradores AVX Base64 SIMD: Encoding/decoding vetorizado UUID v4: Geração com hardware RNG (RDRAND) HTTP Parsing: < 1ms por requisição com AVX2 Network Operations: Hex encoding otimizado

🗂️ Arquitetura

src/ ├── core/ # Infraestrutura (servidor, config, thread pool) ├── crypto/ # Engine criptográfico e assembly otimizado ├── http/ # Protocolo HTTP, roteamento e arquivos estáticos ├── utils/ # Network ops, validation, compression SIMD └── main.cpp # Ponto de entrada e configuração de rotas
