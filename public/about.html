🔧 Sobre o HTTPS Server

Detalhes técnicos e arquitetura do servidor de alta performance

⚙️

Stack Tecnológico

C++17 moderno com recursos avançados
OpenSSL 3.0 para criptografia e TLS
CMake sistema de build cross-platform
NASM assembler x86-64

🔐

Criptografia Assembly

AES-NI implementação hand-optimized
SHA-256 AVX hash com registradores AVX
Custom Provider integração OpenSSL nativa
P-256 curva elíptica otimizada

🚀

Performance

3.51 GB/s throughput AES-NI
Thread Pool detecção automática de cores
Zero-copy Buffer reutilização de memória
AVX2 operações de memória otimizadas

📊 Benchmarks

AES-NI Assembly: 3.51 GB/s (20M blocos, 305 MB em 0.08s) SHA-256 Assembly: Alta performance com registradores AVX HTTP Processing: < 1ms por requisição Memory Usage: Otimizado com buffer reutilizável Concurrent Connections: Múltiplas conexões simultâneas Static File Serving: Alto throughput de entrega

🏗️ Arquitetura

src/ ├── core/ # Infraestrutura (servidor, config, thread pool) ├── crypto/ # Engine criptográfico e assembly otimizado ├── http/ # Protocolo HTTP, roteamento e arquivos estáticos ├── utils/ # Buffer de performance, logging e utilitários └── main.cpp # Ponto de entrada e configuração de rotas
